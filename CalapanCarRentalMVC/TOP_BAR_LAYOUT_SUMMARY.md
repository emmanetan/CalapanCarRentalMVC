## Component Breakdown

### Left Side Components:
1. **Hamburger Menu Button** (`[?]`)
   - Toggles sidebar visibility
   - White text on red background
   - Hover effect with subtle background

2. **Logo Image** (`??`)
   - Company logo
   - Height: 40px (desktop), 30px (mobile)
   - Linked to Home page

3. **Brand Name** ("Calapan Car Rental")
   - Bold, white text
   - Hidden on small mobile screens
   - Linked to Home page

4. **Role Badge** ([Admin] or [Customer])
   - Red badge for Admin
   - Blue badge for Customer
   - Small, uppercase text

### Right Side Components:
1. **Notification Bell Icon** (`??`)
   - Font Awesome bell icon
   - White color
   - Position: Absolute positioning for badge

2. **Notification Badge** (`(3)`)
- Red circular badge
   - Shows unread notification count
   - Position: Top-right of bell icon
   - Visually hidden text for accessibility

3. **Notification Dropdown**
   - Appears when bell icon clicked
   - Min-width: 350px (desktop)
   - Max-height: 500px with scroll
   - Rounded corners, shadow effect
   - Header with count
   - List of notifications
   - "View All" link at bottom

## User Interactions

1. **Click Notification Bell**
   - Opens dropdown
   - Shows list of notifications
   - Badge remains visible

2. **Hover Notification Item**
   - Background changes to light red
   - Left border appears (red)
   - Smooth transition

3. **Click Notification**
   - Navigates to relevant page
   - (Future: Marks as read)

4. **Click "View All"**
   - Goes to full notifications page
   - (Future enhancement)

## Accessibility Features

- `aria-expanded` attribute on dropdown button
- `aria-labelledby` on dropdown menu
- `visually-hidden` class for screen readers
- High contrast colors (white on red)
- Large touch targets (48px minimum)
- Keyboard navigation support
