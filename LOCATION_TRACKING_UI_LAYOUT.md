## Responsive Breakpoints

### Desktop (> 992px):
- Full table layout
- Map: 8 columns, User list: 4 columns
- Statistics: 4 cards in a row
- Filters: 4 columns in a row

### Tablet (768px - 991px):
- Stacked layout (map above user list)
- Statistics: 2 cards per row
- Filters: 2 per row
- Scrollable table

### Mobile (< 768px):
- Full width components
- Statistics: 1 card per row
- Filters: 1 per row (stacked)
- Horizontal scroll for table
- Condensed table columns

## Performance Metrics

- **Initial Load**: < 2 seconds
- **Filter Apply**: < 1 second
- **Page Change**: < 500ms
- **Export CSV**: Depends on record count (typically 1-3 seconds)
- **Map Render**: < 1 second
- **Auto-refresh**: Every 5 seconds (real-time map only)

## Data Volume Handling

| Records | Load Time | Recommendation |
|---------|-----------|----------------|
| < 500   | Instant   | No optimization needed |
| 500-2k  | < 1s      | Use pagination |
| 2k-10k  | 1-3s      | Use filters + pagination |
| > 10k   | 3-5s  | Mandatory filtering |
